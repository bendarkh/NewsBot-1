WordPress Teknoloji Haberi Otomasyonu ve SEO Takip Sistemi

== Description ==

NewsBot Automation, WordPress sitenizde teknoloji haberlerini otomatik olarak yönetmenizi ve SEO performansınızı takip etmenizi sağlayan kapsamlı bir eklentidir.

= Özellikler =

* **Otomatik Haber Analizi**: Teknoloji haberlerini otomatik olarak analiz eder
* **SEO Takibi**: Anahtar kelime sıralamalarınızı takip eder
* **Web Sitesi Analitikleri**: Ziyaretçi istatistiklerini görüntüler
* **Google Trends Entegrasyonu**: Güncel trend konularını takip eder
* **Otomatik İçerik Üretimi**: Trend konulara göre içerik önerileri
* **Detaylı Raporlama**: Kapsamlı performans raporları

= Kurulum Sonrası Yapılacaklar =

1. WordPress admin panelinde "NewsBot" menüsüne gidin
2. Ayarlar bölümünden API anahtarlarınızı girin
3. Takip etmek istediğiniz anahtar kelimeleri ekleyin
4. Google Analytics ID'nizi girin

== Installation ==

1. Plugin dosyalarını `/wp-content/plugins/newsbot-automation/` klasörüne yükleyin
2. WordPress admin panelinde eklentiyi etkinleştirin
3. "NewsBot" menüsünden ayarları yapılandırın

== Frequently Asked Questions ==

= Hangi API anahtarlarına ihtiyacım var? =

* Google Analytics API anahtarı (isteğe bağlı)
* SERP API anahtarı (SEO takibi için)
* Google Trends API anahtarı (isteğe bağlı)

= Eklenti ücretsiz mi? =

Evet, eklenti tamamen ücretsizdir. Ancak bazı API servisleri ücretli olabilir.
